目标功能：提供一个可以快速创建多方块的抽象类 ✅已完成

需要实现：✅所有功能已实现

一。多方块放置类 ✅已完成 - MultiblockPlacer.java
流程为：
1. 玩家手持对应物品
2. 右键其他方块，触发检测
3a. 检测通过，放置整个结构
3b. 检测未通过，撤销放置事件
4a.拦截原版方块放置事件并放置整个结构，主方块应在放置时设置（由 AbstractMultiblock.onRightClickPlace 处理）

二。多方块破坏类 ✅已完成 - MultiblockBreaker.java
流程为：
1. 玩家破坏任意结构方块，触发检测
2a. 为主方块，直接破坏整个结构
2b. 为其他结构方块，通过 NBT 坐标数据重定向到主方块然后破坏整个结构

三。多方块 NBT 读写管理类 ✅已完成 - MultiblockNbtManager.java
需要功能：
1. 多方块主方块的 NBT 读取与写入操作实现与管理

四。多方块右键 GUI 类 ✅已完成 - MultiblockGuiHandler.java
需要功能：
1. 玩家右键主方块时打开 GUI
2. 玩家右键结构方块时通过结构方块内存储的主方块坐标重定向到主方块然后打开 GUI
注：该类会在将来扩展其他功能

五。多方块结构处理类 ✅已完成 - IMultiblockStructure.java, MultiblockStructure.java
注：处理多方块结构内容，使其可用于放置检测与破坏检测
处理流程：
1. 传入结构信息
2. 解析
3. 返回可用的数据（结构信息）

传入的结构信息示例（1.12.2 版本内容，实现时需要修改）：
    protected BlockPattern createStructurePattern() {
        return FactoryBlockPattern.start()
                .aisle("BBBBB", " BBB ")
                .aisle("BBYBB", " BBB ")
                .aisle("BBBBB", " BBB ")
                //B 代表其他结构方块
                .where('B', air())
                // 留空代表可为任意方块
                .where(' ', any())
                // Y 代表主方块
                .where('Y', selfPredicate())
                .build();
    }
注：结构可扩展，非固定长宽高（示例内容为一个底座 5x5，上半部分 3x3，整体为 5x5x2 的结构）
